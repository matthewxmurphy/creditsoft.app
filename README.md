## Features

- **Client Management** - Full client lifecycle management
- **Metro2 Compliance** - 30+ error type detection codes
- **AI-Powered Disputes** - RAG-based intelligent letter generation
- **Automated Drip Campaigns** - Client communication automation
- **CRM Integration** - Built-in customer relationship management
- **Self-Hosted** - Data stays on your servers (intranet ready)

## Tech Stack

- **Framework**: CodeIgniter 4.4
- **Database**: MySQL/MariaDB
- **PDF Generation**: TCPDF
- **Frontend**: Vanilla HTML/CSS/JS

## Installation

1. Upload ZIP to your server
2. Run the installer at `/installer/`
3. Follow the setup wizard
4. Login at `/dashboard`

**Default Login**: `admin@credit.com` / `admin123`

## Quick Start

```bash
# Create deployment ZIP (excludes dev files)
zip -r creditsoft.zip . -x "*.DS_Store" -x "*.git*"
```
