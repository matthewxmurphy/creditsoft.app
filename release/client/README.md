# CreditSoft Client

Client portal version of CreditSoft - Metro2-first credit repair software.

## Purpose

The client-facing portal for credit repair clients to track their progress.

## Tech Stack

- CodeIgniter 4.4
- PHP 8.1+
- MySQL/MariaDB
- TCPDF

## Login

- URL: `/login`
- Email: `admin@credit.com`
- Password: `admin123`

## Installation

1. Upload to server
2. Run installer at `/installer/`
3. Configure database credentials
