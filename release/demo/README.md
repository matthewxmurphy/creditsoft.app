# CreditSoft Demo

Demo version of CreditSoft - Metro2-first credit repair software.

## Purpose

Playground/demo site to showcase the CreditSoft intranet to Ashley.

## Features

- Full credit repair workflow management
- Client management
- Credit report parsing with Metro2 compliance
- Dispute letter generation
- AI-assisted features

## Login

- URL: `/login` or `/dashboard`
- Email: `admin@credit.com`
- Password: `admin123`

## Database Reset

Run `reset_demo.php` via cron to reset database to clean state.

## Tech Stack

- CodeIgniter 4.4
- PHP 8.1+
- MySQL/MariaDB
- TCPDF

## Installation

1. Upload to server
2. Run installer at `/installer/`
3. Configure database credentials
